## How to run this website

* make sure you set the environment to the python environment preparing your backend (app.py) file.
*  The database server used here is the Mysql server/ MySQL workbanch
*  Change the database configerations 
*  open you terminal run this command : "python setup-database.py" to set up the database.
*  then run this command: "python app.py" to start the backend server
*  lastly run the backend-connector.js file to run the frontend
